## 问答题

1. #### 如何在 HTML 文档中嵌入 JavaScript 脚本？

   在HTML页面中插入JavaScript脚本需要使用 `<script></script>` 标签。 `<script></script>` 之间包含了JavaScript代码。

2. #### JavaScript 需要预先编译吗？它是如何执行的？

   JavaScript不需要预先编译。它是解析执行的。

3. #### 如何使用 JavaScript 在网页加载完毕时弹窗？

   在页面文件</body>前插入以下代码

   ```html
   <script>
   alert("弹窗");
   </script>
   ```
